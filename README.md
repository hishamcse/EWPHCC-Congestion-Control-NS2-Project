# EWPHCC-NS2-Project
Exponentially Weighted Probabilistic Hybrid Congestion Control for TCP. This was done for the NS2 Project of Computer Networks Sessional Course

## Consulted Paper
 * [PHCC for TCP](https://github.com/hishamcse/EWPHCC-NS2-Project/blob/master/Report/Consulted%20Paper-%20PHCC%20for%20tcp.pdf)
 
N.B: I have modified some cases and formulas of this paper. The modified formulas, used topology and codes added in the project report and modified files folder
